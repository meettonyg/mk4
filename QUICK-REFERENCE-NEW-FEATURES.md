## 5Ô∏è‚É£ Image Cropping Tool

### Why Crop Images?

**Brand Consistency:**
- Uniform image sizes
- Professional appearance
- Better grid alignment

**Perfect For:**
- Headshots (square crop)
- Banner images (16:9 crop)
- Feature photos (4:3 crop)
- Portfolio work (3:2 crop)

### How to Crop

**Step 1: Upload Image**
1. Upload photo/logo as normal
2. Image appears in editor

**Step 2: Open Cropper**
1. Look for image preview section
2. Click blue "Crop" button
3. Cropper modal opens

**Step 3: Select Aspect Ratio**
Choose from presets:
- **Free** - Any size
- **Square** - Perfect circles, profile pics
- **16:9** - Widescreen, banners
- **4:3** - Standard photos
- **3:2** - Photography standard

**Step 4: Adjust Crop**
- **Drag crop box** to reposition
- **Drag corners** to resize
- **Real-time preview** shows result

**Step 5: Apply**
1. Click "Apply Crop"
2. Image uploads to WordPress
3. Original replaced with cropped version
4. Success notification appears

### Cropper Controls

**Mouse:**
- Click and drag crop box to move
- Click and drag corners to resize
- Click outside crop box to see overlay

**Touch (Mobile):**
- Tap and drag to move
- Pinch corners to resize
- Fully responsive

**Keyboard:**
- ESC to cancel
- Works on all devices

### Pro Tips

**For Logos:**
- Use Square or Free aspect ratio
- Leave white space around logo
- Crop tight for clean look

**For Photos:**
- Use 16:9 for hero images
- Use 4:3 for standard photos
- Use Square for profile pictures

**For Consistency:**
- Use same aspect ratio for all images
- Creates professional grid layout
- Matches brand guidelines

## üéØ Common Workflows

### Setting Up Photo Gallery
1. Upload 3-6 professional photos
2. Add descriptive alt text to each
3. Crop photos to consistent size (4:3 recommended)
4. Drag to arrange best photos first
5. Choose caption style (Overlay recommended)
6. Preview in lightbox

### Setting Up Logo Grid
1. Upload client/partner logos (PNG recommended)
2. Crop logos to square (1:1) for consistency
3. Add company names as alt text
4. Drag to prioritize top clients first
5. Choose "Show Below Logo" or "Hide Names"
6. Preview frontend

### Optimizing for SEO
1. Add unique alt text to every image
2. Use descriptive, keyword-rich text
3. Include location/context when relevant
4. Keep alt text under 125 characters
5. Save and publish media kit
