# Media Gallery

The Media Gallery is your central hub for managing all images, logos, and media files used across your media kits. Upload once, use everywhere.

## Accessing the Media Gallery

### From the Dashboard
1. Go to your **Dashboard**
2. Click **Media Gallery** in the navigation
3. Browse, upload, and organize your files

### From the Editor
1. While editing a media kit, click on any image field
2. Click **Choose from Gallery**
3. Select an existing image or upload a new one

## Uploading Media

### Single File Upload

1. Click **Upload** in the Media Gallery
2. Select a file from your computer
3. Wait for the upload to complete
4. Your file is now available for use

### Bulk Upload

1. Click **Upload**
2. Select multiple files (hold Ctrl/Cmd to select several)
3. Or drag and drop multiple files into the upload area
4. All files upload simultaneously

### Supported File Types

| Type | Formats | Max Size |
|------|---------|----------|
| Images | JPG, PNG, GIF, WebP | 10MB |
| Logos | PNG, SVG | 5MB |
| Documents | PDF | 20MB |

## Organizing Your Media

### Using Folders

Create folders to organize your files:
1. Click **New Folder**
2. Name your folder (e.g., "Headshots", "Logos", "Event Photos")
3. Drag files into folders to organize them

### Suggested Organization


### Linking to Brand Kits

Associate media with specific brands:
1. Select files in the gallery
2. Click **Link to Brand Kit**
3. Choose the appropriate Brand Kit
4. Files will appear in that brand's asset library

## Managing Files

### Viewing File Details

Click on any file to see:
- File name and type
- Dimensions (for images)
- File size
- Upload date
- Where it's used (linked media kits)

### Renaming Files

1. Click on the file
2. Click the **Edit** icon next to the filename
3. Enter a new name
4. Press Enter to save

### Deleting Files

1. Select the file(s) to delete
2. Click **Delete** or the trash icon
3. Confirm deletion

> **Warning:** Deleting a file removes it from all media kits where it's used.

### Replacing Files

Update an image everywhere it's used:
1. Click on the file
2. Click **Replace**
3. Upload the new version
4. All media kits using this file automatically update

## Using Media in Your Media Kit

### Adding Images to Components

1. Click on a component with an image field
2. Click the image placeholder or **Change Image**
3. Choose from:
   - **Media Gallery** - Your uploaded files
   - **Upload New** - Add a new file
   - **External URL** - Link to an image hosted elsewhere

### Image Optimization

Guestify automatically optimizes your images:
- Generates responsive sizes
- Compresses for web performance
- Creates WebP versions for modern browsers

### Recommended Image Sizes

| Use Case | Recommended Size |
|----------|-----------------|
| Profile Photo | 500x500px minimum |
| Hero Background | 1920x1080px |
| Logo | 500px wide (PNG with transparency) |
| Gallery Images | 1200px on longest side |
| Thumbnails | 400x400px |

## Tips for Great Media

### Profile Photos

- Use professional quality images
- Ensure good lighting (natural light is best)
- Choose a clean, uncluttered background
- Show your face clearly
- Smile genuinely

### Logos

- Upload PNG with transparent background
- Include both light and dark versions if needed
- Ensure adequate resolution for all sizes
- Keep file sizes reasonable

### Gallery Images

- Include variety (different settings, angles)
- Provide high-resolution versions for media use
- Add descriptive filenames for easy finding
- Consider including both horizontal and vertical orientations

## Troubleshooting

### Upload Fails

- Check file size (must be under the limit)
- Verify file type is supported
- Try a different browser
- Check your internet connection

### Image Looks Blurry

- Upload a higher resolution version
- Ensure the original isn't already low quality
- Check the display size vs. uploaded size

### Can't Find a File

- Use the search bar to search by filename
- Check different folders
- Use filters to narrow by file type or date
