# Understanding the Interface

This guide helps you navigate the Guestify Media Kit Builder interface so you can work efficiently and make the most of all available features.

## Main Interface Areas

The builder interface is divided into several key areas:

```
┌─────────────────────────────────────────────────────────────┐
│                      TOP TOOLBAR                             │
├──────────────┬──────────────────────────────┬───────────────┤
│              │                              │               │
│   COMPONENT  │       CANVAS / PREVIEW       │    EDIT       │
│   SIDEBAR    │                              │    PANEL      │
│              │                              │               │
│              │                              │               │
└──────────────┴──────────────────────────────┴───────────────┘
```

## Top Toolbar

The toolbar at the top provides quick access to essential functions:

| Button | Function |
|--------|----------|
| **Save** | Manually save your work (auto-save is also enabled) |
| **Preview** | View your media kit as visitors will see it |
| **Theme** | Change the visual theme |
| **Settings** | Access media kit settings and SEO options |
| **Publish** | Make your media kit live |

### View Modes
Switch between different device views to ensure your media kit looks great everywhere:
- **Desktop** - Full-width view
- **Tablet** - Medium-width preview
- **Mobile** - Smartphone preview

## Component Sidebar (Left Panel)

The left sidebar is your component library:

### Component Categories

**Basic Components**
- Biography
- Profile Photo
- Contact

**Content Components**
- Hero
- Topics
- Questions
- Offers

**Social & Media**
- Social Links
- Podcast Player
- Video Intro
- Photo Gallery

**Authority Builders**
- Testimonials
- Stats
- Authority Hook
- Logo Grid (Featured In)

**Branding**
- Company Logo
- Personal Brand Logo

### Adding Components
1. Find the component you want
2. Click and drag it to the canvas
3. Drop it in your desired position
4. Release to place

### Component Accordion Groups
Components are organized into expandable groups:
- Click a group header to expand/collapse
- **Basic** - Essential components everyone needs
- **Premium** - Advanced components for standout media kits

## Canvas (Center Area)

The canvas is your working area where you build your media kit:

### Selecting Components
- **Click** on any component to select it
- Selected components show a blue border
- Drag handles appear for repositioning

### Reordering Components
1. Click and hold the **drag handle** (⋮⋮) on any component
2. Drag up or down to reorder
3. Release to drop in the new position

### Visual Feedback
- **Blue outline** - Currently selected component
- **Green highlight** - Valid drop zone
- **Red highlight** - Invalid drop zone

## Edit Panel (Right Panel)

When you select a component, the right panel shows editing options:

### Content Tab
Edit the actual content of your component:
- Text fields
- Images
- Links
- Lists

### Design Tab
Customize the appearance:
- Layout options
- Spacing
- Alignment
- Visibility settings

### AI Assist
Many components offer AI-powered content generation:
1. Click the **✨ Generate** button
2. Review the generated content
3. Accept, regenerate, or edit manually

## Keyboard Shortcuts

Speed up your workflow with these shortcuts:

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + S` | Save |
| `Ctrl/Cmd + Z` | Undo |
| `Ctrl/Cmd + Shift + Z` | Redo |
| `Delete` / `Backspace` | Remove selected component |
| `Escape` | Deselect / Close panel |

## Auto-Save

Your work is automatically saved every few seconds. You'll see:
- **"Saving..."** - Save in progress
- **"Saved"** - All changes saved
- **"Unsaved changes"** - Changes pending (rare)

> **Note:** If you see "Unsaved changes" for more than a few seconds, click Save manually or refresh the page.

## Tips for Efficient Building

1. **Start with a template** - Don't build from scratch unless you need to
2. **Work top to bottom** - Add components in the order visitors will see them
3. **Use Preview often** - Check your work frequently on different devices
4. **Let AI help** - Use the AI tools to generate content quickly
5. **Save your brand kit** - Set up colors once and reuse everywhere
