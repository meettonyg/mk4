# Podcast Notes Generator

Create professional show notes from your podcast episodes or appearances.

## Overview

The Podcast Notes Generator helps you create:
- Episode summaries
- Key takeaways
- Timestamps and chapters
- Resource lists
- SEO-optimized descriptions

## Types of Show Notes

### Summary Notes
Brief overview of the episode for quick reference.

### Detailed Notes
Comprehensive breakdown with timestamps and quotes.

### SEO Notes
Optimized for search discoverability.

### Promotional Notes
Formatted for sharing and marketing.

## Show Notes Structure

### Standard Format
```
EPISODE TITLE

EPISODE SUMMARY
2-3 sentence overview

KEY TAKEAWAYS
• Takeaway 1
• Takeaway 2
• Takeaway 3

TIMESTAMPS
00:00 - Introduction
03:00 - Topic 1
...

NOTABLE QUOTES
"Quote from guest" - Guest Name

RESOURCES MENTIONED
• Resource 1: [link]
• Resource 2: [link]

ABOUT THE GUEST
Brief bio

CONNECT WITH GUEST
Links to social, website, etc.
```

## Tips for Great Show Notes

### Start with Key Takeaways
These get the most attention. Make them actionable and specific.

### Include Timestamps
Listeners love being able to jump to specific sections.

### Pull Great Quotes
Use memorable quotes for social sharing and highlights.

### Link Everything
Make it easy for listeners to find mentioned resources.

### Optimize for SEO
Include relevant keywords naturally in the description.
