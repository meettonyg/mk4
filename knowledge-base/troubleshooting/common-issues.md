# Troubleshooting Common Issues

This guide helps you resolve the most common issues you may encounter while using Guestify Media Kit Builder.

## Editor Issues

### Media Kit Won't Load

**Symptoms:**
- Blank screen when opening editor
- Loading spinner that never stops
- Error message about loading failure

**Solutions:**

1. **Refresh the page**
   - Press `Ctrl/Cmd + R` or click the refresh button

2. **Clear browser cache**
   - Chrome: `Ctrl/Cmd + Shift + Delete` → Clear cached images and files
   - Firefox: `Ctrl/Cmd + Shift + Delete` → Cache
   - Safari: Develop menu → Empty Caches

3. **Try a different browser**
   - Recommended: Chrome, Firefox, Safari, Edge (latest versions)

4. **Check your internet connection**
   - Ensure stable internet connectivity

5. **Disable browser extensions**
   - Ad blockers and privacy extensions can interfere
   - Try incognito/private mode

---

### Changes Not Saving

**Symptoms:**
- "Unsaved changes" message persists
- Changes disappear after refresh
- Save button seems stuck

**Solutions:**

1. **Wait for auto-save**
   - Auto-save runs every few seconds
   - Look for "Saved" indicator

2. **Manual save**
   - Click **Save** button or press `Ctrl/Cmd + S`

3. **Check for errors**
   - Look for red error messages
   - Required fields may be empty

4. **Refresh and retry**
   - Refresh the page
   - Make changes again
   - Save manually

5. **Check storage quota**
   - Your account may have reached storage limits

---

### Components Won't Drag

**Symptoms:**
- Components don't respond to drag
- Can't reorder components
- Drag handle doesn't appear

**Solutions:**

1. **Click to select first**
   - Click the component before trying to drag

2. **Use the drag handle**
   - Look for the ⋮⋮ icon to drag

3. **Check for locked components**
   - Some components may be locked in position

4. **Refresh the editor**
   - Try refreshing the page

---

### Preview Not Matching Editor

**Symptoms:**
- Preview looks different from editor
- Styling not appearing in preview
- Layout breaks in preview

**Solutions:**

1. **Save before previewing**
   - Unsaved changes won't appear in preview

2. **Clear browser cache**
   - Old styles may be cached

3. **Check responsive view**
   - Preview may be showing a different device size

4. **Theme may not be applied**
   - Verify theme is saved and applied

---

## Image and Media Issues

### Images Won't Upload

**Symptoms:**
- Upload fails or times out
- Error message during upload
- Image doesn't appear after upload

**Solutions:**

1. **Check file size**
   - Images must be under 10MB
   - Compress large images before uploading

2. **Check file format**
   - Supported: JPG, PNG, GIF, WebP
   - Convert unsupported formats

3. **Check internet connection**
   - Large uploads need stable connection

4. **Try a different browser**
   - Some browsers handle uploads differently

5. **Reduce image dimensions**
   - Images over 4000px may cause issues

---

### Images Appear Blurry

**Symptoms:**
- Uploaded images look low quality
- Images are pixelated on the page

**Solutions:**

1. **Upload higher resolution**
   - Minimum recommended: 1200px on longest side
   - Profile photos: 500x500px minimum

2. **Check original file**
   - If original is low quality, upload won't improve it

3. **Use appropriate image size**
   - Don't stretch small images to fill large areas

---

### Media Gallery Not Loading

**Symptoms:**
- Gallery shows blank
- Files not appearing
- Loading indefinitely

**Solutions:**

1. **Refresh the page**
2. **Check filters**
   - Make sure no filters are hiding your files
3. **Search for files**
   - Use search to find specific files
4. **Clear browser cache**

---

## Account and Access Issues

### Can't Log In

**Symptoms:**
- Login page shows error
- Password not accepted
- Account locked

**Solutions:**

1. **Check credentials**
   - Verify email and password are correct
   - Check for typos

2. **Reset password**
   - Click "Forgot Password"
   - Check email (including spam folder)

3. **Clear cookies**
   - Old session cookies may cause issues

4. **Try incognito mode**
   - Rules out extension interference

---

### Features Missing or Disabled

**Symptoms:**
- Some tools or features aren't visible
- "Upgrade" messages appear
- Limited functionality

**Solutions:**

1. **Check your plan**
   - Some features are plan-specific
   - Review your subscription level

2. **Refresh after upgrade**
   - If recently upgraded, refresh to see new features

3. **Contact support**
   - If features should be available but aren't

---

## AI Tool Issues

### AI Not Generating Content

**Symptoms:**
- Generate button does nothing
- Loading but no results
- Error message appears

**Solutions:**

1. **Fill required fields**
   - AI needs input to generate content
   - Check for empty required fields

2. **Provide more context**
   - Very short inputs may fail
   - Add more details

3. **Try again**
   - Temporary issues may resolve on retry

4. **Check internet connection**
   - AI requires active connection

---

### Generated Content is Poor Quality

**Symptoms:**
- Content is too generic
- Doesn't match your style
- Missing key information

**Solutions:**

1. **Provide better input**
   - More specific details = better output
   - Include numbers, achievements, specifics

2. **Use the refine feature**
   - Click "Refine" with feedback
   - Tell AI what to improve

3. **Try different variations**
   - Generate multiple times
   - Compare and choose best

4. **Edit manually**
   - Use AI as a starting point
   - Customize to your voice

---

## Publishing Issues

### Media Kit Won't Publish

**Symptoms:**
- Publish button fails
- Error during publishing
- Media kit stays in draft

**Solutions:**

1. **Check required content**
   - Some fields may be required before publishing

2. **Save first**
   - Make sure all changes are saved

3. **Check your plan limits**
   - You may have reached published kit limits

4. **Try again**
   - Temporary server issues may resolve

---

### Published URL Not Working

**Symptoms:**
- 404 error when visiting URL
- Old version showing
- Page not found

**Solutions:**

1. **Wait a few minutes**
   - Changes may take time to propagate

2. **Clear cache**
   - Both browser cache and try incognito

3. **Check URL is correct**
   - Copy URL directly from dashboard

4. **Republish**
   - Try publishing again

---

## Performance Issues

### Editor Running Slowly

**Symptoms:**
- Laggy interactions
- Slow typing
- Delayed responses

**Solutions:**

1. **Close other tabs**
   - Free up browser memory

2. **Refresh the page**
   - Clear accumulated memory

3. **Use a modern browser**
   - Chrome, Firefox, Safari latest versions

4. **Check computer resources**
   - Close other heavy applications

5. **Reduce component count**
   - Very long media kits may slow down

---

### Page Takes Long to Load

**Symptoms:**
- Slow initial load
- Images loading slowly
- White screen before content

**Solutions:**

1. **Optimize images**
   - Compress large images
   - Use appropriate sizes

2. **Reduce embedded content**
   - Many videos/embeds slow pages

3. **Check internet speed**
   - Run a speed test

---
